01 — Overview

Carousel Ad Strategy

The "TYM to ROLL" carousel is designed using a proven 5-slide storytelling framework: Hook → Problem → Solution → Proof → CTA. Each slide builds on the last, guiding prospects from awareness to action.

Why carousels work: Facebook carousel ads see 72% higher click-through rates than single-image ads. The swipe interaction keeps users engaged longer, and the multi-frame format lets you tell a complete story.

Campaign Objectives

  • Primary: Drive website traffic to inventory pages
  • Secondary: Generate phone calls and lot visits
  • Tertiary: Build brand awareness in Stevens County

05 — Campaign Setup

Facebook Ads Manager

Audience Targeting

  • Age: 35-65
  • Location: Stevens County + 50-mile radius
  • Interests: Farming, Tractors, Rural Living, Agriculture, Livestock, Land Management
  • Behaviors: Farmland owners, Rural area residents

Campaign Settings

Setting Recommendation
Objective Traffic or Conversions
Placements Facebook Feed, Instagram Feed, Stories
Budget $25-50/day for carousel, $15-25/day for geofencing
Duration Run continuously, increase during spring (Feb-May)
Bid Strategy Lowest cost (automatic)
06 — Metrics to Track

Measure What Matters

  • CTR (Click-Through Rate): Target: 1.5%+ for carousel, 2%+ for geofencing
  • CPC (Cost Per Click): Target: Under $2.00
  • Reach: Target: 10,000+/month
  • Frequency: Target: 3-5x per user
  • Landing Page Views: Track actual website visits vs. clicks

Manual tracking: Ask every customer: "How did you hear about us?" Track answers in a spreadsheet to connect ad spend to actual sales.

07 — Quick Start Checklist

Launch in 30 Minutes

  • Download carousel slides from TYM to ROLL campaign page
  • Create new campaign in Facebook Ads Manager
  • Set objective to "Traffic" with website destination
  • Define audience: 35-65, Stevens County +50mi, farming interests
  • Upload 5 carousel slides in order
  • Add headlines and descriptions for each slide
  • Set link to jjriggsequipment.com/tym-tractors
  • Set daily budget at $30-50
  • Enable "Advantage+ Placements"
  • Publish and monitor after 48 hours
